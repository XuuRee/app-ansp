# Bachelor thesis.
[ansp-beta.herokuapp.com](https://ansp.herokuapp.com)

## Requirements

Functional requirements:
1. Task managment:
   - create,
   - edit,
   - finish,
   - assign co-worker  
2. Prioritization of tasks.
3. Repeating tasks.
4. Collaboration:
   - adding comments,
   - adding files,
   - deleting files,
5. Searching.
6. Authentication.

Non-functional requirements:
1. Web application.
2. Simple UI.
3. Implemented in [Django](https://www.djangoproject.com/).
4. Keep It Simpe, Stupid.
